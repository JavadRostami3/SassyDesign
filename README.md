# SassyDesign

A modern, responsive static website built with Sass and JavaScript, featuring a dark mode toggle.

## Development

Install dependencies:

```bash
npm install
```

Compile Sass (watch mode):

```bash
npm run sass
```

Then open `index.html` in your browser.

## Notes

- Sass entrypoint: `scss/styles.scss`
- Output CSS: `css/styles.css`
